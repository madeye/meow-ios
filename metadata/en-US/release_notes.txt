Public beta on iOS 26.

• Subscriptions: tap the pencil icon next to the refresh button to open a built-in YAML editor for any subscription
• Settings → About → Memory now shows live PacketTunnel memory usage
• Stability and signing improvements for the App Store / TestFlight distribution path

Known limitation: non-DNS UDP traffic is not yet forwarded by the tunnel. WireGuard outbounds will not pass traffic, and QUIC/HTTP3 falls back to TCP HTTP/2 (usually transparent to the user). Fix is in progress.

Issues and feedback: https://github.com/madeye/meow-ios/issues
