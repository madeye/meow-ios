meow

A native iOS Mihomo proxy client. Clash-style YAML subscriptions, rule-based routing, DoH, and a Liquid Glass UI on iOS 26.

Join the TestFlight public beta → Firebase Ad Hoc (direct install) View on GitHub

Requires iOS 26 or later (iPhone and iPad). Bring your own Mihomo / Clash subscription — meow does not provide proxy servers.

TestFlight is the recommended channel for most users — install Apple's TestFlight app, tap the link, done. Firebase Ad Hoc is a smaller-batch alternative that installs the app directly through Safari (no TestFlight app, no 90-day rebuild for installed devices); it requires a one-time device registration step and is capped at ~100 active devices.

What it does

  • Clash-protocol YAML subscriptions, with auto-refresh and a built-in YAML editor.
  • Rule-based traffic routing (domain, IP, GeoIP, GeoSITE).
  • DNS over HTTPS resolution.
  • Per-app proxy groups, latency tests, selector switching.
  • On-device packet tunnel — no external server handles your traffic.

Supported protocols

  • Shadowsocks
  • Trojan
  • VLESS
  • WireGuard
  • TUIC
  • Hysteria2

Privacy

meow does not collect, transmit, or share usage data. Subscription URLs, configurations, and traffic stay on your device. The packet tunnel runs locally inside the Network Extension sandbox; the only network traffic is the proxy traffic you configure yourself. See the privacy policy.

Public beta

Some features are still under active development. Known limitation: the tunnel does not yet forward non-DNS UDP traffic — WireGuard outbounds will not pass traffic, and QUIC / HTTP3 falls back to TCP HTTP/2. Please report issues via GitHub Issues or the in-app TestFlight feedback button.
